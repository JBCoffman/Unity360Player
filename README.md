# Unity360Player

A lightweight Unity project for testing and experimenting with 360° video or immersive environments in Unity.

## 🎮 Features

- First-person mouse look with `SimpleMouseLook.cs`
- Clean, minimal Unity project setup
- GitHub-ready with `.gitignore` for Unity

## 🛠 Requirements

- Unity 2022.3.62f1 LTS
- Visual Studio 2022
