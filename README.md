# Unity360Player

A lightweight Unity project for testing and experimenting with 360° video or immersive environments in Unity.

## Features

- First-person mouse look with `SimpleMouseLook.cs`, controls camera perspective.
- Clean, minimal Unity project setup


## Requirements

- Unity 2022.3.62f1 LTS
- Visual Studio 2022

## Getting Started

   git clone https://github.com/JBCoffman/Unity360Player.git

## Controls 

- B to browse for file
- Spacebar for pause/play
- Left/Right Arrow for skip back/forward
- Mousewheel for zoom in/out
